01 // Architecture

1D-Convolutional Neural Network

The model processes raw time-series data from the bike's IMU. Unlike 2D CNNs for images, PotholeNet uses 1D filters to recognize temporal patterns—distinguishing a pothole's "sharp" impact from a speed bump's "rounded" curve.

Model: "PotholeNet_v1"
-------------------------------------------
Layer 1: Conv1D (64 filters, kernel=3, activation='relu')
Layer 2: MaxPooling1D (pool_size=2)
Layer 3: Flatten & Dense (128, dropout=0.5)
Layer 4: Output (Softmax, 2 classes)
