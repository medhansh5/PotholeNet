PotholeNet

Classifying Road Surface Anomalies using Deep Learning & Accelerometry

Abstract

PotholeNet is a neural network designed to identify road surface degradation in real-time. By processing high-frequency vibrations from mobile sensors, the model distinguishes between standard engine noise, expansion joints, and hazardous potholes with a target accuracy of 90%+.

Model Architecture

The core of PotholeNet uses a 1D Convolutional Neural Network (CNN) to extract features from time-series accelerometer data.
