# 🔬 PotholeNet: Production Documentation

This branch (`gh-pages`) hosts the technical whitepaper and model performance metrics for **PotholeNet**, an AI-driven road surface classifier.

### 🌐 Live Technical Site
View the model architecture and training metrics here:
**[https://medhansh5.github.io/PotholeNet/](https://medhansh5.github.io/PotholeNet/)**

---

## 📊 Documentation Scope
Unlike the `main` branch which contains the raw training scripts (TensorFlow/Keras) and datasets, this branch is dedicated to:
* **Model Explainability:** Visualizing the 1D-CNN layers and feature extraction process.
* **Validation Metrics:** Hosting the Confusion Matrix, Precision/Recall curves, and Loss history.
* **API Reference:** Documentation for integrating the PotholeNet model with external telemetry systems like ShadowMap.

## 🛠 Tech Stack (Documentation)
* **Tailwind CSS:** For the minimalist research-grade UI.
* **GitHub Pages:** For global edge deployment.
* **Termux:** Used for mobile deployment and remote git management.
